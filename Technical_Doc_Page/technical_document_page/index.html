What is SQL?

Structured Query Language is what SQL stands for and is a scripting database language used to access, insert, delete or update data in SQL Databases.

SQL databases are primarily called RDBMS or Relational Databases Management Systems.

Advantages of SQL

There are many advantages to developing and running a SQL database to organize your Data including:

  • SQL databases are vertically stable
  • SQL databases are an ideal choice for the complex query intensive
  • Should be used over other database methods when data validity is super important
  • Best option when you need to solve ACID problems and need to support dynamnic queries
Disadvantages of SQL

There are some times when you Don't want to use a SQL RDBMS Database and may opt for another solution, such as NoSQL. Here are a few of those instances where SQL might not work for you:

  • Large amounts of data that FACEBOOK or GOOGLE take in will suffer in performance with a SQL System and need the flexibility of a NoSQL system.
  • SQL databases are not suitable for hierarchical data storage
  • NoSQL databases can be document based, key-value pairs or graph databases where SQL databases are solely table based.
